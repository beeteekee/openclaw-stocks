# Top3股票自动选股系统使用说明

## 系统概述

Top3股票自动选股系统是基于**养家心法V9.5**评分系统的智能选股工具,每天自动分析沪深A股,选出综合得分最高的3只股票供参考。

### 核心功能

✅ **自动选股**: 每天17:00自动运行,选出Top3股票
✅ **智能过滤**: 自动排除北交所和科创板
✅ **综合评分**: 基于基本面、技术面、情绪面多维度评分
✅ **赢面评估**: 提供赢面率和仓位建议
✅ **实时展示**: 前端页面自动展示Top3股票

## 评分系统说明

### 综合得分计算

**综合得分 = 短期得分×60% + 中期得分×30% + 长期得分×10%**

#### 长期得分（10%）
- 只看**行业成长系数**
- 高成长(1.0) = 100分
- 中成长(0.7) = 70分
- 低成长(0.3) = 30分
- 无成长(0.1) = 10分

#### 中期得分（30%）
- 题材热度指数×50% + 财务质量×30% + 技术面×20%
- 题材热度 = 涨停热度×30% + 催化逻辑×30% + 行业成长×40%

#### 短期得分（60%）
- 技术面×80% + 情绪周期×20% - 市值扣分
- 技术面 = 250日均线得分×50% + 缠论买点得分×50%

### 赢面评估

**赢面 = (综合得分 / 100) × 市值系数 × 情绪周期系数**

#### 市值系数（市值越大,赢面越低）
- < 50亿: 1.0
- 50-100亿: 0.98
- 100-200亿: 0.95
- 200-300亿: 0.92
- 300-500亿: 0.88
- 500-1000亿: 0.82
- 1000-3000亿: 0.75
- 3000-5000亿: 0.68
- > 5000亿: 0.60

#### 情绪周期系数
- 绝望: 0.5（极高风险）
- 退潮: 0.7（高风险）
- 高潮: 0.6（中等风险）
- 发酵: 0.8（中低风险）
- 启动: 1.0（低风险,黄金期）
- 冰点: 0.9（低风险,拐点期）
- 中性: 0.75

### 仓位建议

**基础仓位（基于赢面）**
- 赢面 < 50%: 空仓
- 赢面 50-60%: 极小仓试错（≤5%）
- 赢面 60-70%: 小仓（≤20%）
- 赢面 70-80%: 中仓（30%-50%）
- 赢面 80-90%: 中高仓（60%-80%）
- 赢面 > 90%: 重仓（80%-100%）

**市值调整（大市值降低仓位）**
- < 100亿: ×1.5（可提高仓位）
- 100-200亿: ×1.2（适当提高）
- 200-500亿: ×1.0（正常）
- 500-1000亿: ×0.8（降低）
- 1000-3000亿: ×0.6（大幅降低）
- > 3000亿: ×0.5（极低）

## 定时任务

### Cron任务详情

系统默认设置每天17:00自动运行选股脚本:

```bash
0 17 * * * /Users/likan/.openclaw/workspace/run_top3_daily.sh
```

### 修改运行时间

如果需要修改运行时间:

```bash
# 编辑crontab
crontab -e

# 修改时间格式: 分 时 日 月 周
# 例如: 改为每天9:30运行
# 0 9 30 * * * /Users/likan/.openclaw/workspace/run_top3_daily.sh

# 保存并退出
```

### Cron时间格式说明

```
* * * * * command
│ │ │ │ │
│ │ │ │ └─ 星期几 (0-7, 0和7都代表周日)
│ │ │ └─── 月份 (1-12)
│ │ └───── 日期 (1-31)
│ └─────── 小时 (0-23)
└───────── 分钟 (0-59)
```

**常用示例:**
- `0 9 * * *` - 每天9:00
- `30 17 * * *` - 每天17:30
- `0 */6 * * *` - 每6小时
- `0 9 * * 1-5` - 工作日早上9点

## 常见问题

### Q1: 前端页面看不到Top3数据?

**A:** 检查以下几点:
1. 确保stock_service.py服务正在运行
2. 刷新浏览器页面
3. 检查top3_today_result.csv文件是否存在且有数据
4. 查看浏览器控制台是否有错误信息

### Q2: 17:00没有自动更新?

**A:** 检查以下几点:
1. 查看cron任务是否设置: `crontab -l`
2. 查看运行日志: `cat top3_today.log`
3. 确保系统时间正确: `date`
4. 手动运行测试: `./run_top3_daily.sh`

### Q3: Top3股票包含北交所或科创板?

**A:** 系统会自动过滤,但建议检查:
1. 打开top3_today.py文件
2. 找到过滤代码:
```python
df_filtered = df_latest[
    ~df_latest['ts_code'].str.endswith('.BJ') &
    ~df_latest['ts_code'].str.startswith('688.')
]
```
3. 确认代码存在且正确

### Q4: 如何查看历史Top3数据?

**A:** 当前系统只保存最新的Top3数据,如果需要历史数据:
1. 在top3_today.py中添加备份逻辑
2. 每次运行前备份旧数据
3. 使用Git提交历史追溯

### Q5: 综合得分和赢面率不一致?

**A:** 这是正常的:
- **综合得分**: 反映股票的内在质量（基本面+技术面+情绪面）
- **赢面率**: 考虑市值和情绪后的操作成功率

示例:
- 小盘股: 综合得分80分, 赢面率可能75%
- 大盘股: 综合得分80分, 赢面率可能只有60%
