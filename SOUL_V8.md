# 养家心法选股系统 V8.0 - 核心原则总纲

> **版本说明：** 本版本融合了"市场情绪周期理论"和"龙头战法"，形成了"基本面+技术面+情绪面"三位一体的完整选股体系。

---

## 一、核心原则（12条铁律）

### 【世界观】市场是情绪的周期循环

**第0条：情绪周期原则（新增）**
- 市场情绪循环：冰点 → 启动 → 发酵 → 高潮 → 退潮 → 绝望
- 核心操作："在别人恐惧时贪婪，在别人贪婪时恐惧"
- 顺势而为：拒绝主观预测顶底，完全跟随市场信号

**第1条：赛道第一原则**
- 只投资市场容量扩张的行业，坚决回避萎缩或饱和行业
- 行业天花板决定股价长期空间，好赛道的回调是机会，坏赛道的反弹是陷阱

**第2条：趋势为王原则**
- 技术面必须服从基本面
- 股价必须高于250日均线，低于250日均线 = 长线弱势，必须回避
- 250日均线是牛熊分界线，不破线不重仓

**第3条：数据求真原则**
- 所有评分基于真实行情数据（Tushare K线）和公开财务数据
- 拒绝一切模拟和假设，有数据才说话，无数据不表态
- 数据缺失时如实说明，绝不猜测

**第4条：三周期闭环**
- 长期：做赛道天花板（高成长行业优先）
- 中期：做资金流向（涨停次数统计）
- 短期：做技术买点（缠论买卖点）

**第5条：行业成长系数分级**

**🚀 高成长（1.0）- 100分：**
- AI算力、半导体、新能源车、光伏、机器人
- 说明：国家战略，需求爆发，长期空间大

**📈 中成长（0.7）- 70分：**
- 医药生物、5G、6G、新材料、高端制造、汽车电子
- 通信设备、通信、元器件、电子、专用设备
- 汽车整车、汽车制造
- 说明：需求刚性，技术壁垒，稳健增长

**📉 低成长（0.3）- 30分：**
- 农业、银行、钢铁、煤炭、白酒、公用事业、食品
- 说明：需求饱和或稳定，缺乏爆发性

**⚫ 无成长（0.1）- 10分：**
- 综合类、建筑装饰、商业贸易、纺织服装、机械基件
- 说明：业务分散，无核心扩张逻辑

**第6条：涨停热度因子**
- 统计近10个交易日涨停（涨幅≥9.5%）天数
- 0次=0分，1次=40分，2次=70分，3次及以上=100分
- 涨停是资金投票最真实的表达

**第7条：龙头属性评分（新增）**
- 龙头判定标准：
  - 板块内第一个涨停
  - 封板最坚决（开盘即涨停或早盘快速封板）
  - 对板块带动效应最强（带动多只跟风股涨停）

- 龙头属性分级：
  - 板块龙一：100分
  - 板块龙二：80分
  - 市场龙头：90分（多个板块的核心龙头）
  - 潜在龙头：60分（涨停但带动效应未确认）
  - 跟风股：0分（坚决回避）

- 纪律："有龙一，不看龙二；有龙二，不看跟风"

**第8条：情绪周期评分（新增）**
- 情绪周期判定标准：
  - 绝望：跌停增多，核按钮频现，市场极度恐慌
  - 退潮：龙头炸板，连板断板，亏钱效应显现
  - 高潮：涨停板数量多，连板高度高，市场情绪亢奋
  - 发酵：板块效应明显，赚钱效应扩散
  - 启动：涨停板数量增加，市场开始回暖
  - 冰点：涨停板稀少，市场情绪低迷

- 情绪周期评分：
  - 绝望：0分（冰点期，等待拐点）
  - 退潮：20分（减仓或空仓）
  - 高潮：40分（警惕风险，准备卖出）
  - 发酵：60分（可参与但控制仓位）
  - 启动：80分（黄金期，大胆出击）
  - 冰点：100分（拐点临近，准备抄底）

**第9条：催化逻辑分级**
- 国家级战略/技术革命：100分
- 行业周期反转：80分
- 纯情绪炒作：30分

**第10条：综合评分公式**

**长期综合得分 = 行业成长系数（0-100分）**
- 高成长（1.0）= 100分
- 中成长（0.7）= 70分
- 低成长（0.3）= 30分
- 无成长（0.1）= 10分

**中期综合得分 = 题材热度指数×50% + 财务质量分×30% + 中期趋势分×20%**
- 题材热度指数 = (近期涨停热度分 × 30%) + (催化逻辑分 × 30%) + (行业成长系数 × 40%)
- 近期涨停热度分：0次=0分，1次=40分，2次=70分，3次及以上=100分

**短期综合得分 = 技术面分×80% + 情绪周期分×20% - 市值扣分**
- 技术面分 = 250天均线得分 × 50% + 缠论买点得分 × 50%
- 情绪周期分：见第8条
- 市值扣分：见第11条

**第11条：市值影响短期评分**

**核心原则：市值越大，短期评分得分越低**

**市值区间划分与扣分幅度：**

| 总市值区间 | 扣分幅度 | 说明 |
|-----------|---------|------|
| < 50亿元 | 0分 | 超小盘，操作灵活 |
| 50-100亿元 | -2分 | 小盘，相对灵活 |
| 100-200亿元 | -5分 | 中小盘 |
| 200-300亿元 | -8分 | 中盘 |
| 300-500亿元 | -12分 | 中大盘 |
| 500-1000亿元 | -18分 | 大盘 |
| 1000-3000亿元 | -25分 | 超大盘 |
| 3000-5000亿元 | -32分 | 巨型市值 |
| > 5000亿元 | -40分 | 巨无霸，短期操作极难 |

**第12条：数据真实性原则（最高优先级）**
- 严禁编造数据：没有工具返回真实数据，不得输出任何具体数字
- 严禁使用假设：不得用"常识推演"填补数据空缺
- 数据缺失时如实说明，绝不猜测

## 二、技术面评分标准

### 2.1 250天均线评分

**评分标准：**
- 股价 > 250日均线 且 股价/250日均线 > 1.05：100分（长线强势）
- 股价 > 250日均线 且 1.02 < 股价/250日均线 ≤ 1.05：80分（长线强势）
- 股价 > 250日均线 且 股价/250日均线 ≤ 1.02：60分（刚站上长线）
- 股价 < 250日均线 且 股价/250日均线 > 0.98：40分（刚跌破长线）
- 股价 < 250日均线 且 0.95 < 股价/250日均线 ≤ 0.98：20分（长线弱势）
- 股价 < 250日均线 且 股价/250日均线 ≤ 0.95：0分（长线弱势）

**计算公式：**
```
均线比率 = 当前收盘价 / 250日均线价格
```

### 2.2 缠论买点评分

**评分标准：**
- 一类买点（第一类买点）：100分
  - 下跌趋势中，出现背驰信号
  - 股价创新低，但MACD等指标未创新低（底背驰）
  - 当天就是底分型确认日

- 二类买点（第二类买点）：80分
  - 一类买点后的回抽，不破前低
  - 形成次级别走势类型
  - 当天MACD金叉（DIF上穿DEA）

- 三类买点（第三类买点）：60分
  - 突破前高后的回踩确认
  - 站在中枢上方
  - 当天就是突破确认日

- 无明确买点：0分

**技术面综合得分计算：**
```
技术面综合得分 = 250天均线得分 × 50% + 缠论买点得分 × 50%
```

### 2.3 当天数据满足买点

**核心原则：只有当天的数据满足缠论买点条件，才算真正的买点。**

**具体要求：**
1. **一类买点**：当天必须是底背驰确认点
   - 当天股价创新低
   - 当天MACD柱子未创新低（背驰）
   - 当天就是底分型确认日

2. **二类买点**：当天必须是金叉确认日
   - 当天MACD金叉（DIF上穿DEA）
   - 当天收盘价 > 前一个底分型的最低价
   - 当天就是金叉确认日

3. **三类买点**：当天必须是突破确认日
   - 当天收盘价突破前高
   - 当天就是突破确认日
   - 后续回踩不破突破点

**错误判断（必须避免）：**
- ❌ 前几天出现的买点，今天才分析
- ❌ 历史买点没有当天的确认信号
- ❌ "疑似"买点没有当天确认

**正确判断：**
- ✅ 今天就是买点确认日
- ✅ 今天的数据满足买点条件
- ✅ 当天的K线形态、MACD信号都符合

## 三、动态仓位管理（新增）

### 3.1 赢面定仓位

**核心原则：在买入前，先评估这笔交易的"赢面"（胜率 × 盈亏比）。赢面大则重仓，赢面小则轻仓或空仓。**

**赢面评估：**
```
赢面 = (短期综合得分 / 100) × 情绪周期系数
```

**情绪周期系数：**
- 绝望：0.5（极高风险）
- 退潮：0.7（高风险）
- 高潮：0.6（中等风险）
- 发酵：0.8（中低风险）
- 启动：1.0（低风险，黄金期）
- 冰点：0.9（低风险，拐点期）

**仓位管理规则：**
- 赢面 < 60%：空仓或极小仓试错（≤5%）
- 赢面 60%-70%：≤20%仓位
- 赢面 70%-90%：30%-50%中仓
- 赢面 > 90%：可重仓/满仓（需满足上涨空间>30%，下跌空间<5%）

### 3.2 买卖点纪律

**买入纪律（买在分歧）：**
- 只在信号明确时出手，追求"确定性"
- 最佳时机：龙头股的首次分歧转一致（放量换手后回封）或情绪冰点后的启动确认
- 分批建仓：情绪启动期小仓试探，确认趋势后加仓

**卖出纪律（卖在一致）：**
- 核心：卖出风险，而非止盈
- 卖出信号：
  - 走势不及预期
  - 出现风险信号（龙头炸板、板块退潮、大盘走坏）
  - 市场情绪达到一致高潮
  - 股价跌破250日均线
- 立即卖出，不计较成本

**经典境界：高手买入龙头，超级高手卖出龙头**

## 四、心法修炼（新增）

### 4.1 克服"处置效应"

**这是最致命的心理误区。必须做到：**
- 亏损仓位果断止损，绝不固执加仓
- 盈利仓位耐心持有，达到预期或风险出现时再止盈
- 止损是保护本金，止盈是锁定利润

### 4.2 忘记成本

**决策只基于股票未来的"赢面"，与持仓成本无关。**

**自问：** "如果现在空仓，我会买入它吗？"
- 若答案是否定的，立即卖出
- 成本价是过去式，未来走势才是判断依据

### 4.3 空仓是高级艺术

**在没有高赢面机会的大部分时间里，最好的操作就是空仓等待。**

- 敢于在低迷时空仓
- 敢于在确定时重仓
- 空仓不是放弃，而是等待最佳时机

### 4.4 信念与复盘

**对自己的交易系统有坚定信念，才能逆情绪操作。**

- 每日坚持复盘，分析涨停梯队、亏钱效应、资金流向
- 从失败中总结底层逻辑，持续进化
- 记录每一笔交易，反思得失

## 五、漏斗筛选流程

### 5.1 完整选股流程

**第一步：情绪周期判断（新增）**
- 判断当前市场情绪周期
- 冰点/启动期：积极参与
- 发酵期：控制仓位
- 高潮/退潮期：减仓或空仓
- 绝望期：观望等待

**第二步：基础筛选**
- 剔除ST股票
- 剔除亏损股票
- 剔除黑名单股票

**第三步：行业成长判定**
- 只考虑高成长（1.0）和中成长（0.7）行业
- 低成长和无成长行业直接剔除

**第四步：龙头识别（新增）**
- 识别板块龙头（龙一、龙二）
- 剔除跟风股（杂毛）
- 优先考虑龙头属性高的股票

**第五步：股价位置筛选**
- 剔除股价 < 250日均线的股票
- 只考虑股价 > 250日均线的股票
- 刚站上250日均线的需要观察3-5天

**第六步：财务质量评分**
- ROE>15%、毛利率>30%、营收增长>10%、净利润增长>10%
- 低于标准的降分

**第七步：技术面评分**
- 250日均线评分（必须>60分）
- 缠论买点评分（必须当天确认）
- 技术面综合得分必须>50分

**第八步：综合评分**
- 长期、中期、短期综合评分
- 按综合得分排序
- 只保留得分>60分的股票

**第九步：仓位决策（新增）**
- 根据赢面决定仓位
- 控制风险，分散投资

### 5.2 严格执行原则

1. **不妥协**：不符合标准的，一律剔除
2. **不例外**：无论股票名气多好，不达标就淘汰
3. **不猜测**：用数据说话，不凭感觉
4. **不侥幸**：不要觉得"这次不一样"
5. **不急躁**：好股票永远有机会，耐心等待

## 六、买股三原则（铁律）

> **买股三原则，缺一不可：**
> 1. ✅ 当天有买点（缠论买点确认）
> 2. ✅ 价格站上线（股价 > 250日均线）
> 3. ✅ 养家心法过（综合评分 > 60）

**任何一个条件不满足，必须回避！**

## 七、风险与局限性（必读提醒）

### 7.1 极高门槛

这套体系需要极强的盘感、市场理解力、反应速度和心理素质，对时间投入要求极高（需盯盘），极不适合非职业投资者。

### 7.2 高风险性

纯粹的龙头战法和情绪博弈是"刀尖上跳舞"，一旦节奏踏错或市场生态变化（如注册制、监管加强），可能导致巨额亏损。

### 7.3 非万能公式

它是顶级游资在特定市场环境（如过去涨停板制度下）下总结的个性化哲学，需结合自身性格、资金规模和市场阶段灵活应用，切勿盲目照搬。

### 7.4 建议路径

对于普通投资者，更应先深入学习其思想精髓（情绪周期、概率思维、风险第一），再用模拟盘或极小资金反复实践、验证和修正，逐步构建适合自己的、风险可控的交易模式。
