## 配色方案选择

### 为什么选择同花顺风格？

1. **符合中国股市习惯**：
   - 红绿配色是中国股民最熟悉的
   - 绿色=上涨/收益，红色=下跌/风险
   - 用户一看就懂

2. **专业可信**：
   - 白色背景干净专业
   - 颜色对比强烈
   - 传达可靠的品牌形象

3. **易于理解**：
   - 无需解释，符合直觉
   - 降低用户学习成本
   - 提高使用效率

4. **广受欢迎**：
   - 东方财富、雪球等主流应用使用
   - 用户接受度高
   - 降低换色风险

### 2. 底部统计信息功能增强

**新增项**：
```html
<div class="stats-item">
    <div class="stats-label">总查询次数</div>
    <div class="stats-value" id="statsTotalCount">0</div>
</div>
```

**JavaScript逻辑**：
```javascript
// 加载统计数据
async function loadStats() {
    try {
        const response = await fetch('/api/stats');
        if (response.ok) {
            const data = await response.json();
            const stockQueries = data.stock_queries || {};
            
            // 计算总查询次数和股票数量
            let totalCount = 0;
            for (const code in stockQueries) {
                totalCount += (stockQueries[code].count || 0);
            }
            
            const stockCount = Object.keys(stockQueries).length;
            const today = new Date().toISOString().split('T')[0];

            statsDate.textContent = today;
            statsStockCount.textContent = stockCount;
            statsTotalCount.textContent = totalCount.toLocaleString();
        } else {
            console.error('获取统计数据失败');
        }
    } catch (error) {
        console.error('加载统计数据失败:', error);
    }
}
```

**计算逻辑**：
- 遍历`stock_queries`对象
- 累加每个股票的`count`值
- 使用`toLocaleString()`格式化数字（千分位分隔）
