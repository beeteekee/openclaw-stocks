# 2026-02-17 修复宁德时代行业分类错误

## 问题描述
宁德时代（300750.SZ）在选股分析中被错误归类为"电气设备"（无成长行业），导致长期综合得分只有10分，远低于其实际价值。

## 根本原因
代码只使用了Tushare的`industry`字段进行行业匹配，而宁德时代的真实题材在`concept`（概念板块）字段中：

**industry字段：** 电气设备（传统行业，不在高成长列表）

**concept字段：** 磷酸铁锂、储能、特斯拉、蔚来汽车概念、宁德时代概念（新能源相关）

## 修复方案

### 1. 更新INDUSTRY_GROWTH表
新增新能源概念到高成长（1.0）分类：
- 锂电、锂电池、磷酸铁锂
- 储能、新能源
- 特斯拉、比亚迪、蔚来、小鹏、理想

### 2. 修改get_industry_growth_coefficient函数
```python
def get_industry_growth_coefficient(industry, concepts=None):
    """
    优先匹配概念板块中的高成长概念
    如果概念没有匹配，再匹配行业字段
    """
    if concepts:
        for concept in concepts:
            for key, value in INDUSTRY_GROWTH.items():
                if key in concept:
                    return value

    industry = str(industry)
    for key, value in INDUSTRY_GROWTH.items():
        if key in industry:
            return value

    return 0.1
```

### 3. 获取概念板块数据
在分析股票前，先获取概念板块：
```python
concepts = []
try:
    concept_df = pro.concept_detail(ts_code=stock_code)
    if len(concept_df) > 0:
        concepts = concept_df['concept_name'].tolist()
except Exception as e:
    print(f"⚠ 获取概念板块失败: {e}")

growth_coeff = get_industry_growth_coefficient(stock_info['industry'], concepts)
```

## 核心教训
Tushare的`industry`字段是传统行业分类，而`concept`字段更能反映股票的真实题材和概念。对于科技、新能源等新兴行业，必须优先匹配概念板块，否则会误杀优质成长股。
