# 股票分析系统 - 养家心法

基于 **Tushare 真实数据** 的股票分析系统，严格按照 **养家心法10条核心原则** 进行选股分析。

## 系统特点

✅ **真实数据**：使用 Tushare API 获取真实行情和财务数据
✅ **养家心法**：严格执行养家心法10条核心原则
✅ **多维度评分**：
   - 行业成长判定（赛道第一）
   - 财务质量评分（ROE、毛利率、增长率）
   - 技术面评分（250日均线、缠论买点）
   - 催化逻辑评分（题材热度）
   - 综合评分（长期、中期、短期）

## 快速启动

### 方法1：使用启动脚本（推荐）


2. 输入股票代码或名称
   - 推荐格式：`002594.SZ`（包含后缀）

3. 点击"开始分析"
4. 等待30-60秒获取完整分析结果

## 分析标准

### 养家心法10条核心原则

**第1条：赛道第一原则**
- 只投资市场容量扩张的行业
- 行业天花板决定股价长期空间

**第2条：趋势为王原则**
- 技术面必须服从基本面
- 用资金流向（涨停次数）验证基本面逻辑

**第3条：数据求真原则**
- 所有评分基于真实行情数据
- 拒绝一切模拟和假设

**第4条：三周期闭环**
- 长期：做赛道天花板
- 中期：做资金流向
- 短期：做技术买点

**第5条：行业成长系数分级**
- 🚀 高成长（1.0）：AI算力、半导体、新能源车、光伏、机器人
- 📈 中成长（0.7）：医药生物、5G/6G、新材料、高端制造
- 📉 低成长（0.3）：农业、银行、钢铁、煤炭、白酒、公用事业
- ⚫ 无成长（0.1）：综合类、建筑装饰、商业贸易

**第6条：涨停热度因子**
- 统计近10个交易日涨停（涨幅≥9.5%）天数
- 0次=0分，1次=40分，2次=70分，3次及以上=100分

**第7条：综合评分公式**
- 长期综合得分 = 行业成长系数（0-100分）
- 中期综合得分 = 题材热度指数×50% + 财务质量分×30% + 技术面分×20%
- 短期综合得分 = 技术面分×60% + 题材热度指数×40% - 市值扣分

**第8条：催化逻辑分级**
- 国家级战略/技术革命 = 100分
- 行业周期反转 = 80分
- 纯情绪炒作 = 30分

## 故障排除

### 问题：服务器无法启动

**解决方案**：
1. 检查Python版本：`python3 --version`
2. 安装依赖：`pip3 install flask flask-cors tushare pandas numpy`
3. 查看日志：`cat stock_server.log`

### 问题：分析超时

**解决方案**：
1. 检查网络连接
2. 检查 Tushare API 是否正常
3. 尝试使用不同的股票代码

### 问题：页面无法访问

**解决方案**：
1. 确认服务器正在运行：`ps aux | grep stock_server_v2`
2. 检查端口是否被占用：`lsof -i :5000`
3. 重启服务器：`./stop_stock_server.sh && ./start_stock_system.sh`
