## 核心策略

### 养家心法三铁律
1. **情绪周期**：冰点→启动→发酵→高潮→退潮→绝望
2. **龙头属性**：有龙一，不看龙二；有龙二，不看跟风
3. **数据求真**：有数据才说话，无数据不表态

### 数据源策略
- **行情数据**：Tushare API
- **舆情数据**：东方财富股吧（真实数据，不用模拟）
- **分析算法**：基于规则+机器学习的混合方案

### 风险控制
- 每次修改立即提交到Git
- 关键功能必须有降级方案
- 数据异常时及时告警

## 待解决问题

### 优先级：高
1. **队列处理失败**：聊天队列中的对话无法执行，需要排查Gateway配置
2. **LLM超时**：请求频繁超时，需要优化超时设置或模型选择

### 优先级：中
1. **舆情数据量**：当前获取数量有限，需要评估是否足够
2. **前端展示**：舆情分析结果尚未集成到Web界面

### 优先级：低
1. **飞书集成**：暂时禁用，等网络环境改善后重新启用
2. **性能优化**：分析速度和资源占用优化
