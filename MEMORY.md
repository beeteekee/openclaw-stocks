# 养家心法核心原则（V10.2.1）

### 三大铁律
1. **情绪周期**：冰点→启动→发酵→高潮→退潮→绝望
2. **龙头属性**：有龙一，不看龙二；有龙二，不看跟风
3. **数据求真**：有数据才说话，无数据不表态

### 买股三原则
1. ✅ 当天有买点（缠论买点确认）
2. ✅ 价格站上线（股价 > 250日均线）
3. ✅ 养家心法过（综合评分 > 60）

### 行业成长系数
- 高成长（1.0）: AI算力、半导体、新能源车、光伏、机器人、锂电、储能、集成电路、航空航天、生物医药、低空经济、未来能源、量子计算、具身智能、脑机接口、6G、特斯拉、比亚迪、蔚来、小鹏、理想
- 中成长（0.7）: 医药生物、5G、6G、新材料、高端制造、汽车电子、通信设备、通信、元器件、电子、专用设备、汽车整车、汽车制造
- 低成长（0.3）: 农业、银行、钢铁、煤炭、白酒、公用事业、食品
- 无成长（0.1）: 综合类、建筑装饰、商业贸易、纺织服装、机械基件

### 综合评分公式
- 长期得分 = 行业成长系数 × 100
- 中期得分 = 题材热度×50% + 财务质量×30% + 中期趋势×20%
- 短期得分 = 技术面×80% + 情绪周期×20% - 市值扣分
- 综合得分 = 短期×60% + 中期×30% + 长期×10%

### 市值扣分（影响短期得分）
| 总市值 | 扣分 |
|-------|------|
| <50亿 | 0 |
| 50-100亿 | -2 |
| 100-200亿 | -5 |
| 200-300亿 | -8 |
| 300-500亿 | -12 |
| 500-1000亿 | -18 |
| 1000-3000亿 | -25 |
| 3000-5000亿 | -32 |
| >5000亿 | -40 |

### 赢面评估与仓位
- 赢面 = (综合得分/100) × 市值系数 × 情绪周期系数
- 市值系数: 超小盘1.0 → 巨无霸0.60
- 情绪系数: 绝望0.5 → 启动1.0 → 冰点0.9
- 仓位: 赢面<50%空仓，50-60%小仓，60-70%中仓，>90%重仓

# 技术面评分

### 250日均线评分
- 股价>250线且>1.05倍: 100分
- 股价>250线且1.02-1.05倍: 80分
- 股价>250线且≤1.02倍: 60分
- 股价<250线且>0.98倍: 40分
- 股价<250线且0.95-0.98倍: 20分
- 股价<250线且≤0.95倍: 0分

### 缠论买点评分
- 一类买点（底背驰确认）: 100分
- 二类买点（金叉确认）: 80分
- 三类买点（突破确认）: 60分
- 无明确买点: 0分

### 技术面综合得分
技术面 = 250日均线得分×50% + 缠论买点得分×50%

# 漏斗筛选流程

1. 基础筛选: 剔除ST、亏损、黑名单
2. 行业判定: 只考虑高成长（1.0）和中成长（0.7）
3. 龙头识别: 优先龙一、龙二，剔除跟风
4. 股价筛选: 股价必须>250日均线
5. 财务评分: ROE>15%、毛利率>30%、营收增长>10%
6. 技术评分: 250日均线>60分，缠论买点>0分
7. 综合排序: 按综合得分降序排序
8. 赢面评估: 根据赢面决定仓位
9. 舆情分析: 对每日精选10只股票进行舆情分析并按热度排序
