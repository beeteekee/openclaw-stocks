## 工作原则

1. **数据真实性**：所有分析基于真实数据，不伪造、不猜测
2. **异常处理**：所有外部调用都要有try-except，失败时返回友好的错误信息
3. **性能优化**：减少重复API调用，合理使用缓存
4. **日志记录**：重要操作记录到日志，便于调试和监控
5. **版本兼容**：保持与MEMORY.md中最新版本的逻辑一致

4. 验证数据格式是否符合预期
