## 主要改进

### 1. ✅ 真正的缠论买点分析

**之前的版本**：简化版缠论判断（仅基于MACD金叉）

**现在的版本**：严格执行缠论理论，识别三类买点

#### 一类买点（100分）
- **定义**：下跌趋势中，出现底背驰
- **判断逻辑**：
  1. 识别股价的转折点（底分型）
  2. 检查是否有底背驰：股价创新低，但MACD柱子未创新低
  3. 确认当前是否在底背驰点附近（5天内）

#### 二类买点（80分）
- **定义**：第一类买点后的回抽，不破前低，金叉确认
- **判断逻辑**：
  1. MACD出现金叉（DIF上穿DEA）
  2. 检查是否处于下跌后的反弹阶段
  3. 股价未超过近期高点

#### 三类买点（60分）
- **定义**：突破前高后的回踩确认，站在中枢上方
- **判断逻辑**：
  1. 计算最近20日的中枢（最高价和最低价）
  2. 当前股价站在中枢上方
  3. 近期有过回踩但不破中枢下沿

## 技术细节

### 缠论背驰检查算法
```python
def check_divergence(df, macd_df, trend='down'):
    """检查背驰
    
    底背驰：股价创新低，但MACD柱子未创新低
    顶背驰：股价创新高，但MACD柱子未创新高
    """
    if trend == 'down':
        # 找出最近2个最低点
        recent_lows = df.nsmallest(3, 'low').tail(2)
        low1, low2 = recent_lows.iloc[0], recent_lows.iloc[1]
        
        if low1['low'] < low2['low']:  # 价格创新低
            macd1 = macd_df[...]['macd'].values[0]
            macd2 = macd_df[...]['macd'].values[0]
            
            if macd1 > macd2:  # MACD未创新低
                return True, low1['trade_date'], '底背驰'
```

### 转折点识别算法
```python
def find_pivot_points(df, n=5):
    """识别转折点（顶底分型）"""
    # 顶分型：左右低点都低于当前高点
    df['is_top'] = (df['high'] > df['high_prev']) & \
                   (df['high'] > df['high_next'])
    
    # 底分型：左右高点都高于当前低点
    df['is_bottom'] = (df['low'] < df['low_prev']) & \
                      (df['low'] < df['low_next'])
```
