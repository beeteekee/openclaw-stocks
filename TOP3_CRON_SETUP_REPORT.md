## 技术实现细节

### 1. 选股逻辑
- 从沪深A股中筛选涨幅前300名股票
- 排除北交所（.BJ）和科创板（688开头）
- 基于以下维度综合评分:
  - **长期得分（10%）**: 行业成长系数
  - **中期得分（30%）**: 题材热度指数 + 财务质量 + 技术面
  - **短期得分（60%）**: 技术面 + 情绪周期 - 市值扣分

### 2. 赢面评估（V9.5）
```
赢面 = (综合得分 / 100) × 市值系数 × 情绪周期系数

市值系数:
- < 50亿: 1.0
- 50-100亿: 0.98
- 100-200亿: 0.95
- 200-300亿: 0.92
- 300-500亿: 0.88
- 500-1000亿: 0.82
- 1000-3000亿: 0.75
- 3000-5000亿: 0.68
- > 5000亿: 0.60

情绪周期系数:
- 绝望: 0.5
- 退潮: 0.7
- 高潮: 0.6
- 发酵: 0.8
- 启动: 1.0
- 冰点: 0.9
- 中性: 0.75
```

### 3. 仓位建议
- 赢面 < 50%: 空仓
- 赢面 50-60%: 极小仓试错（≤5%）
- 赢面 60-70%: 小仓（≤20%）
- 赢面 70-80%: 中仓（30%-50%）
- 赢面 80-90%: 中高仓（60%-80%）
- 赢面 > 90%: 重仓（80%-100%）

再根据市值调整:
- 小盘（<100亿）: 可提高仓位（×1.2-1.5）
- 中盘（100-500亿）: 正常仓位（×1.0）
- 大盘（>500亿）: 降低仓位（×0.5-0.8）

## 注意事项

1. **定时任务依赖系统时间**: cron任务会在每天17:00自动运行,请确保系统时间正确

2. **数据更新时间**: 选股脚本使用的是Tushare的数据,数据通常在收盘后15:00-17:00之间更新

3. **服务必须运行**: 前端页面显示Top3数据需要stock_service.py服务运行

4. **市场状态**: Top3股票基于当天收盘数据,建议在收盘后(15:00后)查看

5. **风险评估**: Top3股票仅供参考,不构成投资建议,请结合自身情况谨慎决策
