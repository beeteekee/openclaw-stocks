### 舆情热度分解读

**舆情热度分 = 讨论热度40% + 情绪倾向30% + 龙头关注度30%**

#### 讨论热度（0-100分）
- 100分：连续涨停（≥3次），市场关注度极高
- 80分：两次涨停，热门
- 60分：一次涨停，较热门
- 30分：无涨停，一般热度

#### 情绪倾向（0-100分）
- 100分：正面情绪主导（涨停密集+高成长）
- 80分：正面情绪（有涨停+高成长）
- 60分：中性偏正面（有涨停）
- 40分：中性（无涨停表现）

#### 龙头关注度（0-100分）
- 100分：市场龙头（连续涨停）
- 90分：潜在龙头（多次涨停+高成长）
- 70分：板块内关注（有涨停+高成长）
- 50分：一般关注（有涨停）
- 30分：跟风股（无涨停）

#### 舆情级别
- 🔥 爆热（≥80分）：连续涨停+高成长，市场关注度极高
- 📈 热门（60-79分）：有涨停+高成长，有一定关注度
- 😐 一般（40-59分）：中性表现，关注度一般
- ❄️ 冷门（<40分）：无涨停+低成长，关注度低

### 短期得分公式变化

**V9.0旧公式：**
```
短期得分 = 技术面分×80% + 市场情绪周期分×20% - 市值扣分
```

**V10.0新公式：**
```
短期得分 = 技术面分×70% + 舆情热度分×20% + 题材热度指数×10% - 市值扣分
```

**权重调整：**
- 技术面：80% → 70%（仍然是核心）
- 舆情热度：0% → 20%（新增，反映个股市场情绪）
- 题材热度：40% → 10%（从短期降到辅助）
- 市场情绪周期：20% → 0%（移到赢面评估中）

### 常见问题

**Q1：舆情热度分为什么是固定的？**
A：当前舆情数据基于涨停情况和行业成长推断，尚未接入真实舆情API。这是MVP阶段的简化实现。

**Q2：如何接入真实舆情API？**
A：需要调用东方财富股吧、雪球、微博等平台的API，获取实时讨论量、情感分析、热搜排名等数据。

**Q3：舆情热度分的权重为什么是20%？**
A：技术面仍然是核心（70%），舆情热度是辅助（20%），题材热度是参考（10%）。这个权重分配可以根据市场环境调整。

**Q4：舆情热度分和市场情绪周期分的区别？**
A：舆情热度分反映个股的市场关注度和情绪倾向，市场情绪周期分反映整体市场的情绪阶段（冰点、启动、发酵、高潮、退潮、绝望）。
