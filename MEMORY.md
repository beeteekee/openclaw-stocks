## 2026-02-12 市值影响短期评分标准

### 核心原则

### 市值对短期评分的影响

**核心原则：市值越大，短期评分得分越低**

**原因：**
1. 大市值股票短期操作难度大，资金推动困难
2. 小市值股票更容易被短期资金拉动
3. 短期交易需要考虑流动性和资金容量

**市值区间划分与短期评分调整：**

| 总市值区间 | 扣分幅度 | 说明 |
|-----------|---------|------|
| < 50亿元 | 0分 | 超小盘，操作灵活 |
| 50-100亿元 | -2分 | 小盘，相对灵活 |
| 100-200亿元 | -5分 | 中小盘 |
| 200-300亿元 | -8分 | 中盘 |
| 300-500亿元 | -12分 | 中大盘 |
| 500-1000亿元 | -18分 | 大盘 |
| 1000-3000亿元 | -25分 | 超大盘 |
| 3000-5000亿元 | -32分 | 巨型市值 |
| > 5000亿元 | -40分 | 巨无霸，短期操作极难 |

**短期评分计算：**
```
短期综合得分 = (技术面分 × 60% + 题材热度指数 × 40%) - 市值扣分
```

**严格性要求：**
1. 不得使用"近似"或"简化"的市值区间划分
2. 必须精确计算总市值（单位：亿元）
3. 扣分必须严格按照表格执行，不得随意调整
4. 短期评分最低为0分

**影响案例：**
- 002594.SZ比亚迪（8311亿元）：扣分40分，短期综合得分从78.8降至38.8
- 002342.SZ巨力索具（164亿元）：扣分5分，短期综合得分从64.2降至59.2
- 300762.SZ上海瀚讯（250亿元）：扣分5分，短期综合得分从58.8降至53.8

### 严格执行标准的重要性

1. **避免误导**：简化标准会导致对大市值股票的误判
2. **真实反映**：短期操作难度是客观存在的事实
3. **风险提示**：大市值股票的短期交易风险更高

- 股价 < 250日均线 且 股价/250日均线 ≤ 0.95：0分（长线弱势）

**计算公式：**
```
均线比率 = 当前收盘价 / 250日均线价格
```

### 缠论买点评分

  - 形成次级别走势类型
  
- 三类买点（第三类买点）：60分
  - 突破前高后的回踩确认
  - 站在中枢上方

## 2026-02-14 模型三点核心优化

### 优化1：判断当天数据是否满足缠论买点

**核心原则：**
只有当天的数据满足缠论买点条件，才算真正的买点。

**具体要求：**
1. **一类买点**：当天必须是底背驰确认点
   - 当天股价创新低
   - 当天MACD柱子未创新低（背驰）
   - 当天就是底分型确认日

2. **二类买点**：当天必须是金叉确认日
   - 当天MACD金叉（DIF上穿DEA）
   - 当天收盘价 > 前一个底分型的最低价
   - 当天就是金叉确认日

3. **三类买点**：当天必须是突破确认日
   - 当天收盘价突破前高
   - 当天就是突破确认日
   - 后续回踩不破突破点

**错误判断（必须避免）：**
- ❌ 前几天出现的买点，今天才分析
- ❌ 历史买点没有当天的确认信号
- ❌ "疑似"买点没有当天确认

**正确判断：**
- ✅ 今天就是买点确认日
- ✅ 今天的数据满足买点条件
- ✅ 当天的K线形态、MACD信号都符合

   - 需要观察3-5天确认站稳

3. **高于250日均线 = 可关注**

## 2026-02-19 V9.0版本重大升级：综合得分权重优化

### 更新原因
V8.0版本虽然形成了"基本面+技术面+情绪面"三位一体的选股体系，但缺乏一个统一的综合得分。对于短线交易者来说，短期指标（技术面+情绪周期）应该是最重要的，因为短线交易的核心是"买在分歧，卖在一致"，技术买点和情绪周期直接决定了买入时机。

### 核心原则更新

**V9.0核心原则：短线权重 > 中线权重 > 长线权重**

**权重分配：**
- 短期得分（技术面+情绪周期）：60%
- 中期得分（题材热度+财务质量+技术面）：30%
- 长期得分（行业成长）：10%

**核心逻辑：**
1. **短期决定买点**：技术面（250日均线+缠论买点）和情绪周期（市场情绪阶段）决定了当前的买入时机，这是短线交易最关键的因素
2. **中期决定趋势**：题材热度（涨停热度+催化逻辑+行业成长）和财务质量决定了中期趋势的持续性
3. **长期决定赛道**：行业成长系数决定了股票的长期空间和天花板

### 综合得分公式

**V9.0综合得分公式：**
```
综合得分 = 短期得分×60% + 中期得分×30% + 长期得分×10%
```

**各维度得分：**

1. **长期得分（0-100分）：**
   - 只看行业成长系数：行业成长系数×100
   - 高成长（1.0）= 100分
   - 中成长（0.7）= 70分
   - 低成长（0.3）= 30分
   - 无成长（0.1）= 10分

2. **中期得分（0-100分）：**
   - 中期得分 = 题材热度指数×50% + 财务质量分×30% + 技术分×20%

3. **短期得分（0-100分）：**
   - 短期得分 = 技术面分×80% + 情绪周期分×20% - 市值扣分
   - 技术面分 = 250日均线得分×50% + 缠论买点得分×50%
   - 情绪周期分：冰点100分、启动80分、发酵60分、高潮40分、退潮20分、绝望0分

### 综合评级标准

**基于综合得分的评级：**
- **优秀**：综合得分 ≥ 75分
- **良好**：综合得分 ≥ 60分
- **一般**：综合得分 < 60分

**与V8.0的区别：**
- V8.0：基于"长期>=70 and 短期>=60"判断优秀，基于"长期>=60 or 短期>=60"判断良好
- V9.0：直接基于综合得分判断，更科学、更直观

### 权重设计的科学依据

**为什么短期权重最高（60%）？**
1. **短线交易的核心是时机**：技术买点和情绪周期直接决定了当前的买入时机
2. **市场是情绪的周期循环**：短期涨跌由持币者与持筹者的群体博弈决定，而非基本面
3. **"买在分歧，卖在一致"**：最佳买入时机是分歧转一致的时候，这完全由技术面和情绪周期决定

**为什么中期权重中等（30%）？**
1. **题材热度决定资金流向**：涨停热度、催化逻辑、行业成长决定了资金的持续流入
2. **财务质量决定安全性**：ROE、毛利率、营收增长、净利润增长决定了公司的基本面
3. **技术面验证趋势**：中期技术面验证趋势的持续性

**为什么长期权重最低（10%）？**
1. **赛道天花板是底层逻辑**：行业成长决定了股票的长期空间，但对短线交易影响较小
2. **短线交易不追求长期持有**：短线交易目标是快速获利，而非长期价值投资
3. **避免过度关注长期**：过度关注长期可能导致错过短期机会，或者固执持有亏损仓位

### 实际应用示例

**案例1：龙头股（高短期得分）**
- 长期得分：70分（中成长行业）
- 中期得分：80分（题材热度高+财务良好）
- 短期得分：90分（技术买点确认+情绪启动期）
- 综合得分 = 90×60% + 80×30% + 70×10% = 54 + 24 + 7 = **85分（优秀）**

- 短期得分：50分（技术面一般+情绪发酵期）
- 综合得分 = 50×60% + 90×30% + 70×10% = 30 + 27 + 7 = **64分（良好）**

**案例3：成长股（高长期得分）**
- 长期得分：100分（高成长行业）

## 2026-03-10 V10.0版本：舆情分析系统集成

### 更新背景
V9.0版本虽然形成了"基本面+技术面+情绪面"三位一体的综合评分体系，但情绪周期只反映市场整体情绪，缺乏对个股舆情热度、关注度、情绪倾向的量化分析。个股的舆情热度是市场情绪的直接表达，是短线交易的重要参考指标。

### 核心更新

**1. 新增舆情分析模块（sentiment_analysis.py）**

**舆情热度分（0-100分）= 讨论热度40% + 情绪倾向30% + 龙头关注度30%**

- **讨论热度（40%）**：
  - 连续涨停（≥3次）：100分（超级热门）
  - 两次涨停：80分（热门）
  - 一次涨停：60分（较热门）
  - 无涨停：30分（一般热度）

- **情绪倾向（30%）**：
  - 正面情绪主导（涨停密集+高成长）：100分
  - 正面情绪（有涨停+高成长）：80分
  - 中性偏正面（有涨停）：60分
  - 中性：40分

- **龙头关注度（30%）**：
  - 市场龙头（连续涨停）：100分
  - 潜在龙头（多次涨停+高成长）：90分
  - 板块内关注（有涨停+高成长）：70分
  - 一般关注（有涨停）：50分
  - 跟风股（无涨停）：30分

**2. 修改短期得分公式（V10.0）**

**旧公式（V9.0）：**
```
短期得分 = 技术面分×80% + 市场情绪周期分×20% - 市值扣分
```

**新公式（V10.0）：**
```
短期得分 = 技术面分×70% + 舆情热度分×20% + 题材热度指数×10% - 市值扣分
```

**权重调整说明：**
- 技术面从80%降到70%：仍然是核心（技术面决定买入时机）
- 舆情热度新增20%：反映个股市场关注度和情绪倾向
- 题材热度新增10%：从原来的40%降为10%，作为辅助指标
- 市场情绪周期分：从20%降为0%，市场整体情绪已通过赢面评估中的情绪系数体现

**3. 集成舆情分析到选股流程**

在analyze_stock.py和stock_service.py中：
1. 在技术面评分后，调用sentiment_analysis.py计算舆情热度分
2. 将舆情热度分传入calculate_comprehensive_score函数
3. 修改短期得分公式，融合舆情热度分
4. API返回完整的舆情分析结果（sentiment字段）

**4. 舆情热度分级**

- **🔥 爆热（≥80分）**：连续涨停+高成长，市场关注度极高
- **📈 热门（60-79分）**：有涨停+高成长，有一定关注度
- **😐 一般（40-59分）**：中性表现，关注度一般
- **❄️ 冷门（<40分）**：无涨停+低成长，关注度低
